Welcome to the Design System Probe Fixture

This fixture exercises font detection by combining heading and body styles within a simple layout. The typography extractor should distinguish these roles clearly.

Explore Components

Typography Coverage

The card text should use the base body style while the heading inherits the display style. Palette extraction should find blues, oranges, and neutral greys.

Palette and Spacing

Spacing tokens should capture 12px, 16px, 24px, and 32px increments along with the CTA padding.
